## 📱 界面元素说明

### 1. 标题区域
```
🚀 桌面应用合集
选择要启动的应用
```
- 显示应用名称和副标题
- 紫色渐变背景
- 居中对齐

### 2. 应用卡片

#### 卡片组成部分
```
┌──────────────────┐
│  [图标]          │  ← 大表情图标
│  [应用名称]      │  ← 粗体标题
│                  │
│  [应用描述]      │  ← 灰色说明文字
```

#### 卡片状态

**未运行状态**
- 白色背景
- 显示"启动"按钮（紫色）
- 鼠标悬停时上浮

**运行中状态**
- 浅绿色背景
- 绿色边框
- 右上角显示"运行中"标签
- 显示"运行中"（灰色禁用）和"停止"（红色）按钮

## 🎯 交互流程

### 启动应用流程

```
1. 打开启动器
   ↓
2. 找到要启动的应用卡片
   ↓
3. 点击"启动"按钮
   ↓
4. 等待应用启动（几秒钟）
   ↓
5. 卡片变为绿色边框，显示"运行中"
   ↓
6. 应用开始工作（如托盘图标出现）
```

### 停止应用流程

```
1. 在启动器中找到运行中的应用
   ↓
2. 点击红色"停止"按钮
   ↓
3. 应用立即停止
   ↓
4. 卡片恢复为白色
   ↓
5. 显示"启动"按钮
```

## 💡 使用技巧

### Tip 1: 快速查看状态
运行中的应用会显示绿色边框，一眼就能看出哪些应用在运行。

### Tip 2: 后台运行
关闭启动器窗口不会停止已运行的应用，它们会继续在后台工作。

### Tip 3: 重新打开启动器
随时可以重新打开启动器查看和管理应用状态。

### Tip 4: 同时运行多个应用
可以同时启动多个应用，它们互不干扰。
