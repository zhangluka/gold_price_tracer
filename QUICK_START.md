# 快速开始指南

## 🚀 立即开始

### 1. 安装依赖（首次使用）

```bash
npm install
```

### 2. 启动应用

```bash
npm start
```

启动后会打开一个漂亮的启动器窗口！

## 📱 使用启动器

### 启动器界面
启动应用后，你会看到一个现代化的启动器界面：

- 🎨 **渐变背景**：紫色渐变设计
- 📦 **应用卡片**：每个应用以卡片形式展示
- 🎯 **一键启动**：点击"启动"按钮即可运行应用
- 🔄 **实时状态**：运行中的应用会显示绿色边框和"运行中"标签

### 操作步骤

1. **启动应用**
   - 在启动器中找到你想使用的应用
   - 点击"启动"按钮
   - 等待应用启动成功

2. **停止应用**
   - 对于正在运行的应用，会显示"停止"按钮
   - 点击"停止"按钮即可关闭应用

3. **关闭启动器**
   - 点击窗口关闭按钮
   - 已启动的应用会继续在后台运行
   - 可以随时再次打开启动器管理应用

## 💰 黄金价格监控使用

### 功能特性
- ✅ 系统托盘实时显示金价
- ✅ 价格涨跌闪烁提示（红涨绿跌）
- ✅ 自动刷新（60秒）
- ✅ 右键菜单查看详情

### 使用步骤

1. **启动监控**
   ```
   在启动器中点击"黄金价格监控"的"启动"按钮
   ```

2. **查看价格**
   ```
   查看系统托盘（Mac: 右上角状态栏）
   显示格式: Au 456.78
   ```

3. **查看详情**
   ```
   右键点击托盘图标
   可以看到：
   - 最新报价
   - 最高价
   - 最低价
   ```

4. **手动刷新**
   ```
   右键菜单 -> 立即刷新
   ```

5. **停止监控**
   ```
   方式1: 右键菜单 -> 关闭监控
   方式2: 在启动器中点击"停止"按钮
   ```

## 🔧 配置修改

### 修改黄金价格刷新间隔

编辑文件：`src/apps/gold-monitor/config.js`

```javascript
module.exports = {
  UPDATE_INTERVAL: 60 * 1000,  // 改为你想要的间隔（毫秒）
  // 例如: 30 * 1000 表示 30 秒
};
```

### 修改启动器窗口大小

编辑文件：`src/common/constants.js`

```javascript
module.exports = {
  LAUNCHER_WINDOW: {
    width: 900,      // 窗口宽度
    height: 600,     // 窗口高度
    minWidth: 800,   // 最小宽度
    minHeight: 500,  // 最小高度
  },
};
```

## ❓ 常见问题

### Q: 启动器关闭后如何重新打开？
A: 再次运行 `npm start` 或双击应用图标。

### Q: 如何完全退出应用？
A: 
- 方法1: 在启动器中停止所有应用，然后关闭窗口
- 方法2: 右键托盘图标选择"退出"
- 方法3: 使用 Cmd+Q (Mac) 或 Alt+F4 (Windows)

### Q: 价格不更新怎么办？
A: 
1. 检查网络连接
2. 右键托盘图标选择"立即刷新"
3. 重启应用

### Q: 找不到托盘图标？
A: 
- Mac: 查看右上角状态栏
- Windows: 查看右下角系统托盘
- 如果被隐藏，展开隐藏图标
